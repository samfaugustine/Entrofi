Quick reference

Habits, distilled

Do

  • Start from /init or a tight CLAUDE.md
  • /clear between unrelated tasks
  • Knowledge to the vault, rules to CLAUDE.md
  • Fan out big searches to subagents
  • Verify before you assert

Don't

  • Dump everything into one giant CLAUDE.md
  • Let one session sprawl across many jobs
  • Duplicate vault knowledge into memory
  • Paste huge files a subagent could summarize
  • Trust stale notes without checking

Context is the budget. Keep the always-on layer small, pull the rest in on demand, and protect the room the task runs in. That one habit is most of the other 80%.
